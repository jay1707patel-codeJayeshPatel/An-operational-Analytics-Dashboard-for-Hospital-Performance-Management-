# An-operational-Analytics-Dashboard-for-Hospital-Performance-Management-
Interactive Power BI dashboard analyzing hospital performance, patient recovery, treatment costs, and doctor efficiency using healthcare analytics.

Project Overview

This project presents an Operational Analytics Dashboard for Hospital Performance Management developed using Power BI.

The dashboard provides insights into hospital operations, doctor performance, patient recovery scores, treatment costs, and hospital stay duration to support data-driven healthcare decision-making.

The goal is to help healthcare administrators monitor operational efficiency and improve hospital performance.

Key Performance Indicators (KPIs)
KPI	Description
Average Age	Average age of patients
Recovery Score	Average patient recovery score
Treatment Cost	Average treatment expenditure
Hospital Stay	Average hospital stay duration
Patient Count	Total number of patients
Dashboard Features
1. Patient Statistics

The dashboard provides:

Average patient age
Total patient count
Hospital stay duration

These metrics help understand patient demographics and care utilization.

2. Doctor Performance Analysis

The dashboard compares:

Doctor-wise performance
Recovery score contribution
Patient handling statistics

This enables hospitals to identify high-performing doctors and improve care quality.

3. Recovery Score Analysis

Recovery score KPIs help measure:

Treatment effectiveness
Patient outcome quality
Clinical performance

Higher recovery scores indicate better treatment outcomes.

4. Treatment Cost Analysis

The dashboard tracks:

Average treatment costs
Cost distribution
Resource utilization

This supports hospital cost management and budgeting.

Key Insights

Some insights from the dashboard include:

Certain doctors demonstrate higher treatment performance

Treatment costs contribute significantly to hospital operations
Average hospital stay provides insight into care efficiency
